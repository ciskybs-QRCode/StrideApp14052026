# Stride — Replit Source Repository

This is the Replit source repository for the **Stride** dance-school management platform. Code is automatically mirrored to [`ciskybs-QRCode/StrideApp14052026`](https://github.com/ciskybs-QRCode/StrideApp14052026) on every push to `main` via the [Sync to GitHub](.github/workflows/sync-to-github.yml) workflow.

The badge above reflects the last sync result in real-time:
- 🟢 **passing** — the most recent push synced successfully
- 🔴 **failing** — the sync is broken; check the [Actions tab](../../actions/workflows/sync-to-github.yml) and any open [`sync-failure` issues](../../issues?q=is%3Aissue+is%3Aopen+label%3Async-failure)

## Workspace

pnpm monorepo · TypeScript · Expo (mobile) · React + Vite (landing) · Express 5 + PostgreSQL (API)

See [`replit.md`](replit.md) for the full stack overview and key commands.
