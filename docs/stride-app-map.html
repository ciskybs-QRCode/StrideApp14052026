Membro / Genitore

Gestisce i figli, le iscrizioni ai corsi, i pagamenti e la sicurezza del ritiro

✅ Può fare

  • Iscrivere i figli ai corsi
  • Pagare le rette con carta/Apple Pay
  • Gestire il Guardian Circle (chi può ritirare i figli)
  • Emettere QR token monouso per il ritiro
  • Vedere lo storico ritiri con audit trail
  • Firmare documenti legali / liberatorie
  • Segnalare assenza figlio
  • Attivare SOS struttura
  • Ricevere allarmi emergenza (bypass silent mode)
  • Gestire la privacy delle foto del figlio
  • Prenotare lezioni private
  • Cambiare associazione (multi-org)

❌ Non può fare

  • Accedere alla lista altri genitori/soci
  • Modificare i dati del corso
  • Approvare pagamenti o fatture
  • Accedere alle info mediche di altri studenti
  • Scannerizzare QR di altri
  • Modificare le impostazioni dell'associazione
  • Vedere dati di fatturazione operatori

Operatore / Istruttore

Gestisce le presenze, le proprie classi, la fatturazione e il rapporto contrattuale

✅ Può fare

  • Scannerizzare QR di check-in studenti
  • Avviare sessione aula e fare appello
  • Gestire le presenze della propria classe
  • Vedere le info mediche/emergenza degli studenti
  • Attivare SOS con protocolli DRSABCD
  • Gestire l'Emergency Pulse per allarmi struttura
  • Inviare fattura mensile automatizzata
  • Firmare il contratto di collaborazione
  • Inviare dimissioni formali con preavviso
  • Dichiarare disponibilità settimanale
  • Scansionare ticket eventi
  • Comunicare assenza (→ cascata automatica sostituto)

❌ Non può fare

  • Vedere le rette/pagamenti dei genitori
  • Modificare i dati dei corsi
  • Approvare le proprie fatture
  • Accedere alle schede di altri operatori
  • Modificare le impostazioni dell'associazione
  • Vedere i dati di altri istruttori
  • Gestire beacons o hardware

Automazioni — Cosa fa l'app senza input umano

TRIGGER: SOS premuto → Notifica push con bypass silent mode a tutti i genitori → Watchdog 60s → Re-notifica chi non ha confermato → Log incidente
TRIGGER: Operatore assente → Cascata sostituto: contatta operatori disponibili in ordine di affinità → Prima risposta "sì" vince → Admin notificato → Genitori avvisati se nessun sostituto trovato entro 5 minuti
TRIGGER: Operatore si dimette → AI Roster Orchestrator attivato per trovare sostituto permanente → Admin notificato → Accesso app disattivato alla data di fine preavviso
TRIGGER: Bambino non ritirato (orario) → Notifica push al genitore → Escalation all'operatore → Log no-show
TRIGGER: Beacon BLE rilevato → Check-in automatico del bambino senza scan manuale → Transizione zona (ingresso → aula → uscita) tracciata → Genitore notificato di entrata/uscita sicura
TRIGGER: Fattura operatore inviata → Admin notificato per approvazione → Approvazione → Stripe payout automatico al conto operatore
TRIGGER: Documento non firmato (promemoria) → Promemoria push periodico finché tutti i documenti non sono firmati → Legal gate blocca l'accesso
TRIGGER: Smart Roster (admin attiva) → AI raccoglie disponibilità operatori → Genera piano orario ottimizzato per l'anno → Admin approva e pubblica
