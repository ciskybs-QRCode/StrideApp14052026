🚨 REVISED UI/UX OVERHAUL: FRESH, BRIGHT & MINIMALIST INITIAL SETUP WIZARD 🚨

Role: Expert Senior UI/UX Mobile Designer & Flutter Frontend Specialist.
Objective: Completely refactor the Initial Setup / Onboarding wizard screen. Eliminate any heavy dark backgrounds or messy, unaligned elements. The interface must be bright, fresh, luminous, and perfectly ordered, using Navy Blue and Gold strictly as elegant branding accents.

VISUAL DESIGN COMPLIANCE & LIGHT GEOMETRY:

1. CANVAS & LIGHT BACKGROUND
- Background Canvas: Strict, clean, and bright light-neutral background (Pure White #FFFFFF or an ultra-light fresh gray #F8F9FA). Absolutely NO solid dark backgrounds or heavy dark canvases.
- Layout Containment: Group the onboarding form components into a beautifully structured, symmetrical layout. If using cards, they should blend seamlessly into the bright canvas with very soft, subtle, clean definition.

2. CRISP BRANDING ACCENTS (NAVY BLUE & GOLD)
- Primary Typography: All main screen titles, section headers, and field labels must be rendered in sharp, high-contrast Deep Navy Blue (#0A1128). This ensures a premium, readable, and professional look without darkening the app.
- Subtitles & Hints: Render in a balanced Slate Gray for a clean, light aesthetic.
- Focused States: Input fields should have a minimalist light gray border by default, which turns into an elegant Premium Gold (#D4AF37) or Deep Navy Blue (#0A1128) outline only when active/focused.

3. GRID-ALIGNED INPUT FIELDS
- Structure: Every text input field must follow a uniform layout: linear, perfectly aligned, with symmetrical internal padding.
- Spacing: Implement strict fixed vertical gaps (e.g., 16dp) between components. Eliminate any chaotic, asymmetric, or random spacing.

4. THE "COMPLETE SETUP & LAUNCH" BUTTON
- Style: A clean, full-width solid action button. 
- Colors Option: Either a solid Deep Navy Blue (#0A1128) button with bold White text, or a Premium Gold (#D4AF37) button with Navy Blue text. Choose the one that keeps the layout looking fresh, clean, and perfectly corporate.
- Shape: Elegant rounded corners matching the input field design.

SCREEN INTEGRITY CONSTRAINTS:
- Screen Containment: All text containers, fields, checkboxes, and buttons must fit strictly within the mobile phone viewport boundaries. Absolutely zero horizontal scrolling, layout clipping, or overlapping text.
- Language: 100% STRICT ENGLISH for all operational text and interface strings.
